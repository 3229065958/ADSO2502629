trabajando con git

  • git init: inicualizar un proyecto en GIT
  • git config --global user.emailidentificar con correo con proposito local
  • git config --global --unset --all user.nameelimina las configuraciones del nombre de usuario en el repositorio local
  • clear:limpiar pantalla terminal
  • cd nombre-directorio:ingresa a un directorio en la terminal
  • cd ..:salir del directorio de la terminal
  • pwdmostrar la ruta del directorio actaul en la terminal
  • lsmuestra la lista de dcontenidos de una direccion
  • git log:mostrar el historial d cambios en el repositorio
  • git add .:agrega todos los archivos modificados al repositorio
  • git status:verificar el estado del repositorio
  • git commit -m "comentario"guardar los cambios realizados en el repositorio con un mensaje resumido
  • git branch mostar el nombre de la rama
  • git branch -m mainpermitecambiar el nombre de la rama
